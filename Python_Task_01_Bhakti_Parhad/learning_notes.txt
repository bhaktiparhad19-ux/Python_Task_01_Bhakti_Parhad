PYTHON TASK 01 - LEARNING NOTES

1. What is Python?
   Python is a high-level, interpreted programming language that is easy to learn and widely used for automation, scripting, web development, data analysis, and cybersecurity.

2. Variables
   Variables are used to store data values in a program.

Example:
name = "Bhakti"
age = 18

3. Input and Output
   The input() function is used to take input from the user.
   The print() function is used to display output on the screen.

Example:
name = input("Enter your name: ")
print("Hello", name)

4. Data Types
   Common Python data types:

* String (str) : Text data
* Integer (int) : Whole numbers
* Float (float) : Decimal numbers
* Boolean (bool) : True or False

5. Arithmetic Operators
   Python supports basic mathematical operations:

* Addition

- Subtraction

* Multiplication
  /  Division
  %  Modulus

Example:
a = 10
b = 5
print(a + b)

6. Conditional Statements
   Conditional statements help make decisions in a program.

Example:
if age >= 18:
print("Adult")
else:
print("Minor")

7. Python Modules
   Modules provide additional functionality.
