# Python Task 01 - System Information & Basic Automation

## Internship

White Band Associates Summer Internship 2026

## Objective

The objective of this task is to learn Python programming fundamentals, user input handling, arithmetic operations, Python modules, and basic automation techniques.

## Project Structure

### system_info.py

Displays:

* Name
* Branch
* Internship Name
* Current Date and Time

### user_input.py

Accepts user input for:

* Name
* Age
* Favorite Technology

Displays a formatted summary based on the entered information.

### calculator.py

A menu-driven calculator that performs:

* Addition
* Subtraction
* Multiplication
* Division

Includes input validation for division by zero.

### challenge.py

Displays system information such as:

* Operating System Name
* Current Username
* Current Working Directory
* Python Version
